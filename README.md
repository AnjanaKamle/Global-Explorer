# Global Explorer - Country & Travel Dashboard

## Project Overview

Global Explorer is an interactive web application designed to help users discover and explore countries around the world. It provides a clean and intuitive dashboard where users can view essential country data such as capital, population, region, currency, and national flags.

This project demonstrates the use of JavaScript, API integration, and dynamic UI development.

## API Used

This project uses the **REST Countries API** to fetch real-time data about countries.

Data includes:

* Country Name
* Capital City
* Population
* Region
* Currency
* Flag

---

## Features

* **Search Functionality**
  Search countries by name or capital in real-time.

* **Region-Based Filtering**
  Filter countries by continents such as Asia, Europe, Africa, etc.

* **Sorting Options**
  Sort countries alphabetically or by population (ascending/descending).

* **Interactive UI**
  Smooth hover effects and dynamic content updates.

* **Responsive Design**
  Fully optimized for mobile, tablet, and desktop devices.

## Setup and Execution

To run this project locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/AnjanaKamle/Global-Explorer.git
   ```

2. **Navigate to the project folder**

   ```bash
   cd Global-Explorer
   ```

3. **Run the project**

   * Open `index.html` in your browser
   * OR use **Live Server** in VS Code for better development experience

> No additional installations or dependencies are required.
